- E algumas outras dependências

## 🚀 Como executar

Em todos os projetos, ao acessar a pasta, é preciso baixar os pacotes com o gerenciador de sua preferencia.

Depois de instalado, é preciso criar um arquivo `.env` na raiz da oasta `server`, e colocar as variáveis de ambiente, seguindo o arquivo `.env.example`.

Por fim, basta usar o comando `yarn dev` para iniciar o servidor em ambos casos, ou `npm run dev` caso prefira usar o npm.
